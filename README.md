# Win-PortFwd
Setup windows port forwarding using native netsh command

### Install:
> git clone https://github.com/deepzec/Win-PortFwd.git


### Usage:
>  .\win-portfwd.ps1

or 

> powershell.exe -noprofile -executionpolicy bypass -file .\win-portfwd.ps1
